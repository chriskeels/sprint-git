# App Features Documentation

## Core Features

1. Secure Login for Associates and Teachers

2. Video Lesson Streaming (Math and Science)

3. Online Quiz System

4. Progress Tracking Dashboard

5. Grade Tracking for Teachers

6. Assignment and Homework Completion

7. Notification System for Updates and Deadlines

---

## System Needs

1. Dockerized Application Environment (consistent across all computers)

2. Docker Compose for App and Database Integration

3. Continuous Integration (CI) for Automatic Testing

4. Database Connectivity (Prisma ORM)

5. Environment Variable Management (dotenv)

6. Secure Authentication System
